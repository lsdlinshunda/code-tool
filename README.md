# 密码工具帮助文档

## 一、程序概述

本密码工具为大一上程序设计课大作业，使用python 2.7和Tkinter编写，用于古典密码的加密与解密。需要系统拥有python 2.7和Tkinter的环境。

实现的功能如下：
#### 1. 支持文本的导入与导出
#### 2. 支持下列密码的加密与解密：
- 替代密码
    - 凯撒密码
    - 维吉尼亚密码
    - 摩斯密码
    - 手机9键
    - 字母表数字
    - 电脑键盘QWE替代
    - 电脑密码V字替代
- 换位密码
    - 栅栏密码
    - 倒叙密码
- 其它
    - 进制转换

## 二、使用说明

执行目录下的```code tool.py```启动程序，程序启动后出现图形界面
 
 ![image](http://chuantu.biz/t6/99/1508140593x3396866884.png)

 #### 界面说明：

### 1. 菜单栏
- 文件菜单  
  文件菜单下有三个子项:
    - "打开"：可从文件中导入文本到界面中央的文本框，若要正常显示中文需要文本使用utf-8编码，但本工具并不支持加密中文。
    - "保存"：可将文本框中的文本保存为txt文件，若文本中有中文，则保存的txt文件使用utf-8编码
    - "退出"：关闭程序退出
- "选择加密方式"菜单  
此菜单可选择加密方式，以打钩的方式表示当前所选加密方式，同时文本框下方也会出现相应的密码名称及其可改属性
- 帮助菜单
    - "使用说明"：打开帮助文件
    - "关于作者"：显示作者的信息
### 2. 文本框
输入和显示文本，自动换行，用鼠标滚轮可以翻页
### 3. 加密选项区域
显示加密方法和可改选项
### 4. 按钮区
- "加密"：用当前加密方法加密文本框内容，并将结果覆盖到文本框中
- "解密"：用当前加密方法解密文本框内容，并将结果覆盖到文本框中
- "倒叙"：将文本框内容倒叙
- "清除"：清除文本框内容
